オブジェクトの基本

オブジェクトの基本

オブジェクトとは関連のあるデータと機能の集合である。

それらは変数と関数で構成される。オブジェクトではそれぞれプロパティとメソッドと呼ばれる


  const person = {
    name: ['Bob', 'Smith'],
    age: 32,
    gender: 'male',
    interests: ['music', 'skiing'],
    bio: function() {
      alert(this.name[0] + ' ' + this.name[1] + ' is ' + this.age + ' years old. He likes ' + this.interests[0] + ' and ' + this.interests[1] + '.');
    },
    greeting: function() {
      alert('Hi! I\'m ' + this.name[0] + '.')
    }
  };

ドットによる記述

ドット記法を用いてアクセスする。

オヴジェクトは名前空間として機能する。オブジェクト内にカプセル化されたものにアクセスするには、まずオブジェクト名を入力する


  person.age
  person.intrests[1]
  preson.bio()

サブ名前空間

オブジェクト内に他のオブジェクトをもつ


name: ['Bio', 'Smith'],

name : {
  first: 'Bob',
  last: 'Smith'
},

person.name.first
person.name.last

角括弧による記述

角括弧でもできる


person.age
person.intrests[1]

person['age']
person['name']['first']
